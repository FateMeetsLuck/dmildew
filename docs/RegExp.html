Class: RegExp

RegExp(pattern, flags)

Regular expressions can be created with the class constructor or regular expression literals such as /foo/gi

Constructor

new RegExp(pattern, flags)

Creates a new RegExp object. Throws an error if the regular expression doesn't compile.
Parameters:
Name Type Description
pattern String The pattern (the text between forward slashes in a literal)
flags String The flags such as 'g' (the text after the last forward slash in a literal)

Members

lastIndex

Sets the index into the string being searched with exec()
