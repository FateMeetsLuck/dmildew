setTimeout(callback, ms, …args) → {*}

Sets a callback to be called after ms number of milliseconds, with args passed as arguments to the callback. Note that this function is asynchronous and timeouts only run after the stack is clear. Unlike setImmediate, this function adds Fibers to the last of the queue.
Parameters:
Name Type Attributes Description
callback function The Function to execute
ms integer The number of milliseconds to wait before calling the Function. This number is only a minimum; there is no guarantee of when the callback is executed.
args * <repeatable>
 The arguments to pass to the callback
Returns:
A Timeout that can be cancelled with clearTimeout.
