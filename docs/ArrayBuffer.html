Class: ArrayBuffer

ArrayBuffer(arg)

This class implements a read-only buffer of raw bytes such as a downloaded media file. To use the data it must be converted to one of the typed arrays such as with `new Uint8Array(someArrayBuffer)`.

Constructor

new ArrayBuffer(arg)

Constructs an ArrayBuffer. If the argument is empty, constructs an empty ArrayBuffer. If the argument is a number, constructs a buffer of the given size. If the argument is a typed array, constructs a buffer of bytes out of the data in the typed array.
Parameters:
Name Type Description
arg * The size of the buffer in bytes or a typed array.
