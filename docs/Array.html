Class: Array

Array

Arrays are a list of values stored in one reference type. Arrays in Mildew can be multidimensional or jagged. Arrays are created with the array expression: comma separated expressions surrounded by brackets. Array elements can be indexed with the brackets [] operator containing a numerical expression. If the expression in brackets is not numerical, it is converted to string and the array is accessed like a regular Object. If an array index exceeds the length of the array minus one, or is a negative number, a ScriptRuntimeException is thrown. There is no constraint on the type of each element of an array, such as there would be with Java generics or templates of lower level languages. The length property of an array can be reassigned to shrink or grow the array. Arrays are passed by reference to functions such that changing individual elements modifies the array parameter, however, reassigning the array parameter has no effect.
