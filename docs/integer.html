Class: integer

integer

This primitive type is an 8-byte signed integer that can hold values between -9,223,372,036,854,775,808 and 9,223,372,036,854,775,807. Integers are created from expressions containing only numbers with no decimal points. It is automatically converted to 8-byte double when used as an operand in an expression containing a double operand. Primitive types are passed by value not reference.
