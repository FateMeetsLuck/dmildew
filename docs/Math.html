Namespace: Math

Math

The Math namespace contains various native D functions powered by D's standard math library that perform mathematical calculations on numerical arguments. Many of these functions convert integers to doubles before performing the calculation, and thus return a double. When given invalid, non-numerical arguments, the result is NaN. Results are in radians and must be manually converted to degrees. Functions that expect a double will automatically convert integer arguments to doubles.
