Class: Function

Function

A Function is a subclass of Object that can be called with the function call expression: any primary expression followed by parentheses containing an optional list of comma separated expressions. All functions defined with the function keyword have a field called "prototype" which becomes an object's __proto__ field when the function is used with the new keyword. Functions may also describe Generators by placing an asterisk (*) after the function keyword. When such a Function is called, the return value is a Generator object with a next() method for iteration through yields. All functions have the following methods.

bind(thisObjopt) → {function}

Bind a "this" object to be associated with the function when called. Note that unlike JavaScript, extra arguments cannot be bound. To unbind a function so that its "this" is natural, simply pass undefined or no arguments to bind. This method returns a new Function instead of modifying the original Function.
