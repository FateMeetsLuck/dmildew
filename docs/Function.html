Class: Function

Function

A Function is a subclass of Object that can be called with the function call expression: any primary expression followed by parentheses containing an optional list of comma separated expressions. All functions defined with the function keyword have a field called "prototype" which becomes an object's __proto__ field when the function is used with the new keyword. All functions have the following methods.

bind(thisObjopt)

Bind a "this" object to be associated with the function when called. Note that unlike JavaScript, extra arguments cannot be bound. To unbind a function so that its "this" is natural, simply pass undefined or no arguments to bind. Note that instead of copying the function, this modifies the function that bind was called on in place, so there is no return value. To get fresh functions with a different binding each time, functions should be generated from an expression that evaluates to a Function object. This may change in the future.
