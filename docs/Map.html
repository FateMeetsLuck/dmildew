Class: Map

Map()

The Map class provides an associative array that allows any type of key, not just Strings, unlike Object. Even Objects and complex types can be keys. Maps cannot be assigned with the array index [] operator, but are assigned with the set method. Likewise, the index operator cannot be used to get Map values, but the get method should be used instead. Maps can be used in for-in loops like a regular Object.

Due to the implementation of opCmp and opEquals in ScriptAny, true and 1 cannot be unique keys; likewise 0.0, null, and 0 cannot be unique keys. It is undefined behavior to modify a key that is a complex object of a Map.
